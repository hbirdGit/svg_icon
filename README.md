# svg_icon

## Add to dependencies

### production:  
`npm install git+https://github.com/hbirdGit/svg_icon.git`

or

`"dependencies": {
    "svg_icons": "github:hbirdGit/svg_icon"
  },`

### Test: 
`npm install "../package"`

or 

`"dependencies": {
    "svg_icons": "file:../package"
  },`

## Importing

`import SVGIcon from 'svg_icons'`
